Panel - Grupo 07

Teoria

Un panel es un contenedor visual que agrupa y organiza contenido relacionado en una interfaz de usuario. Generalmente, los paneles tienen un encabezado que describe su propósito y un cuerpo que contiene el contenido principal. Los paneles son útiles para dividir la información en secciones manejables, mejorando la legibilidad y la navegación dentro de una aplicación o sitio web.

Este panel es un ejemplo de panel simple con un encabezado y un cuerpo.

Para construir un panel html con bootstrap se utiliza la clase .panel junto con otras clases adicionales para definir el estilo y la estructura del panel como lo es el .panel-heading para el encabezado o titulo del panel y el .panel-body en donde va el contenido tambien la clase .panel-footer para el pie del panel.

Ademas existen 5 tipos de paneles segun el uso o finalidad:

  • Panel por defecto: .panel-default
  • Panel de informacion: .panel-info
  • Panel de exito: .panel-success
  • Panel de advertencia: .panel-warning
  • Panel de error: .panel-danger

Partes de un Panel

1. Panel Heading (.panel-heading)

Encabezado que describe el propósito del panel

2. Panel Body (.panel-body)

Contenido principal del panel

3. Panel Footer (.panel-footer)

Pie del panel para información adicional

Ejemplo

Ejemplos de código

Ejemplo 1

En este ejemplo se demuestra la estructura básica de un panel desde su cabecera, cuerpo y pie de panel.

<!-- Ejemplo 1: Estructura básica de un panel -->
<div class="panel panel-default">
    <div class="panel-heading">Título</div>
    <div class="panel-body">Contenido</div>
    <div class="panel-footer">Pie</div>
</div>
Salida del ejemplo 1
Ejemplo 2

Este ejemplo demuestra un panel anidado con una lista dentro del body.

<!-- Ejemplo 2: Panel con lista -->
    <div class="panel panel-default">
        <div class="panel-heading">Panel con lista</div>
        <ul class="list-group">
            <li class="list-group-item">Elemento 1</li>
            <li class="list-group-item">Elemento 2</li>
        </ul>
    </div>
Salida del ejemplo 2
Ejemplo 3

Este código demuestra cómo se puede anidar una tabla dentro del cuerpo del panel para mostrar información relevante.

<!-- Ejemplo 3: Panel con tabla -->
    <div class="panel panel-default">
        <div class="panel-heading">Panel con tabla</div>
        <table class="table">
            <tr><th>Nombre</th><th>Edad</th></tr>
            <tr><td>Ana</td><td>23</td></tr>
            <tr><td>Juan</td><td>30</td></tr>
        </table>
    </div>
Salida del ejemplo 3

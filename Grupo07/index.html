Panel - Grupo 07

Teoria

Un panel es un contenedor visual que agrupa y organiza contenido relacionado en una interfaz de usuario. Generalmente, los paneles tienen un encabezado que describe su propósito y un cuerpo que contiene el contenido principal. Los paneles son útiles para dividir la información en secciones manejables, mejorando la legibilidad y la navegación dentro de una aplicación o sitio web.
Este panel es un ejemplo de panel simple con un encabezado y un cuerpo.
Para construir un panel html con bootstrap se utiliza la clase .panel junto con otras clases adicionales para definir el estilo y la estructura del panel como lo es el .panel-heading para el encabezado o titulo del panel y el .panel-body en donde va el contenido tambien la clase .panel-footer para el pie del panel.
Ademas existen 5 tipos de paneles segun el uso o finalidad:
  • Panel por defecto: .panel-default
  • Panel de informacion: .panel-info
  • Panel de exito: .panel-success
  • Panel de advertencia: .panel-warning
  • Panel de error: .panel-danger
